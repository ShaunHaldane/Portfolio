Shaun Haldane - Software Engineer


Hi I am Shaun, coming from a mechanical engineering background I have been interested in robotics for many years. I started my coding journey working on Arduino projects to supplement my studies at university, and because I really enjoy building products. I used Python for scientific programming and became interested in software engineering methodology. I then started learning Java because of the object oriented approach of the language. I decided in 2019 that I would switch careers and aim to be a full time software engineer. I have been building projects which are shown on this portfolio with the Java Spring stack for about 2 years and feel that I am ready to use my skills and build software solutions professionally.
